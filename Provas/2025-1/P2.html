Gabarito - Segunda Prova de Álgebra Linear

Questão 1 - Subespaços Vetoriais
a) Considere o conjunto \(W = \{(x, y, z) \in \mathbb{R}^3 \mid x - 2y + z = 0\}\). Verifique se o vetor nulo \((0, 0, 0)\) pertence a W.
Ver Resposta

Para verificar se \((0,0,0)\) pertence a W, substituímos \(x=0, y=0, z=0\) na condição do conjunto:

\(0 - 2(0) + 0 = 0 - 0 + 0 = 0\)

Como a igualdade \(0=0\) é verdadeira, o vetor nulo pertence a W.

b) Sejam \(\vec{u} = (x_1, y_1, z_1)\) e \(\vec{v} = (x_2, y_2, z_2)\) dois vetores pertencentes a W. Mostre que a soma \(\vec{u} + \vec{v}\) também pertence a W.
Ver Resposta

Se \(\vec{u} \in W\), então \(x_1 - 2y_1 + z_1 = 0\). Se \(\vec{v} \in W\), então \(x_2 - 2y_2 + z_2 = 0\).

A soma é \(\vec{u} + \vec{v} = (x_1+x_2, y_1+y_2, z_1+z_2)\). Para que a soma pertença a W, ela deve satisfazer a condição:

\((x_1+x_2) - 2(y_1+y_2) + (z_1+z_2) = 0\)

Reagrupando os termos: \((x_1 - 2y_1 + z_1) + (x_2 - 2y_2 + z_2)\)

Pelas nossas hipóteses, sabemos que cada parênteses vale zero: \(0 + 0 = 0\)

Como a condição é satisfeita, W é fechado sob a soma.

c) Seja \(\vec{u} = (x, y, z)\) um vetor em W e \(k\) um escalar real. Mostre que o vetor \(k\vec{u}\) também pertence a W.
Ver Resposta

Se \(\vec{u} \in W\), então \(x - 2y + z = 0\).

O vetor \(k\vec{u}\) é \((kx, ky, kz)\). Para que ele pertença a W, deve satisfazer a condição: \((kx) - 2(ky) + (kz) = 0\)

Fatorando \(k\): \(k(x - 2y + z)\). Pela nossa hipótese, o termo no parêntese é zero: \(k(0) = 0\)

Como a condição é satisfeita, W é fechado sob a multiplicação por escalar.

d) Agora, considere o conjunto \(H = \{(x, y, z) \in \mathbb{R}^3 \mid x - 2y + z = 1\}\). Mostre, usando um contra-exemplo, por que H não é um subespaço vetorial.
Ver Resposta

A maneira mais simples é testar a presença do vetor nulo. Substituindo \((0,0,0)\) na condição de H: \(0 - 2(0) + 0 = 0\). A condição exige que o resultado seja 1, mas obtivemos 0. Como \(0 \neq 1\), o vetor nulo não pertence a H. Portanto, H não é um subespaço vetorial.

Questão 2 - Base, Dimensão e Coordenadas
a) Considere os vetores \(\vec{v_1} = (1, 1, 0)\), \(\vec{v_2} = (0, 1, 1)\) e \(\vec{v_3} = (1, 0, 1)\). Verifique se o conjunto \(S = \{\vec{v_1}, \vec{v_2}, \vec{v_3}\}\) é linearmente independente (LI).
Ver Resposta

Montamos uma matriz com os vetores como colunas e calculamos seu determinante.

$$A = \begin{pmatrix} 1 & 0 & 1 \\ 1 & 1 & 0 \\ 0 & 1 & 1 \end{pmatrix}$$

\(\det(A) = 1(1) - 0 + 1(1) = 2\). Como \(\det(A) \neq 0\), o conjunto S é linearmente independente.

b) O conjunto S forma uma base para o \(\mathbb{R}^3\)? Justifique.
Ver Resposta

Sim. A dimensão do \(\mathbb{R}^3\) é 3. Como temos um conjunto com 3 vetores que é linearmente independente (provado no item a), um teorema garante que este conjunto também gera o \(\mathbb{R}^3\). Portanto, S é uma base para o \(\mathbb{R}^3\).

c) Encontre as coordenadas do vetor \(\vec{w} = (5, 4, 3)\) na base S.
Ver Resposta

Buscamos \(c_1, c_2, c_3\) tais que \(c_1\vec{v_1} + c_2\vec{v_2} + c_3\vec{v_3} = \vec{w}\). Resolvendo o sistema correspondente, encontramos \(c_3 = 2\), \(c_2 = 1\), e \(c_1 = 3\). As coordenadas são \((3, 1, 2)\).

d) Verifique se as coordenadas encontradas estão corretas.
Ver Resposta

Calculamos a combinação linear \(3\vec{v_1} + 1\vec{v_2} + 2\vec{v_3}\):

\(3(1, 1, 0) + 1(0, 1, 1) + 2(1, 0, 1) = (3+0+2, 3+1+0, 0+1+2) = (5, 4, 3)\)

O resultado é o vetor \(\vec{w}\), então a verificação foi bem-sucedida.

Questão 3 - Mudança de Base

Seja a base canônica \(A = \{\vec{e_1}, \vec{e_2}, \vec{e_3}\}\) e a base \(B = \{\vec{b_1}, \vec{b_2}, \vec{b_3}\} = \{(1,0,0), (1,1,0), (1,1,1)\}\) do \(\mathbb{R}^3\).

a) Encontre as coordenadas do vetor \(\vec{v}=(3,2,1)\) na base B resolvendo o sistema \(c_1\vec{b_1} + c_2\vec{b_2} + c_3\vec{b_3} = \vec{v}\).
Ver Resposta

Resolvendo o sistema \(\begin{cases} c_1 + c_2 + c_3 = 3 \\ c_2 + c_3 = 2 \\ c_3 = 1 \end{cases}\), obtemos \(c_3 = 1, c_2 = 1, c_1 = 1\). As coordenadas são \([\vec{v}]_B = \begin{pmatrix} 1 \\ 1 \\ 1 \end{pmatrix}\).

b) Monte a matriz de mudança de base da base B para a base A, \(P_{A \leftarrow B}\), colocando os vetores de B como colunas.
Ver Resposta
$$ P_{A \leftarrow B} = \begin{pmatrix} 1 & 1 & 1 \\ 0 & 1 & 1 \\ 0 & 0 & 1 \end{pmatrix} $$
c) Calcule a matriz inversa \(P_{B \leftarrow A} = (P_{A \leftarrow B})^{-1}\).
Ver Resposta

A matriz inversa é \(P_{B \leftarrow A} = \begin{pmatrix} 1 & -1 & 0 \\ 0 & 1 & -1 \\ 0 & 0 & 1 \end{pmatrix}\).

d) Use a matriz \(P_{B \leftarrow A}\) para encontrar as coordenadas \([\vec{v}]_B\) a partir de \([\vec{v}]_A = \begin{pmatrix} 3 \\ 2 \\ 1 \end{pmatrix}\).
Ver Resposta

\([\vec{v}]_B = P_{B \leftarrow A} \cdot [\vec{v}]_A = \begin{pmatrix} 1 & -1 & 0 \\ 0 & 1 & -1 \\ 0 & 0 & 1 \end{pmatrix} \begin{pmatrix} 3 \\ 2 \\ 1 \end{pmatrix} = \begin{pmatrix} 1 \\ 1 \\ 1 \end{pmatrix}\)

e) Compare os resultados dos itens (a) e (d). O que isso nos diz?
Ver Resposta

Os resultados são idênticos. Isso nos diz que multiplicar um vetor pela matriz de mudança de base inversa (\(P_{B \leftarrow A}\)) é um atalho que realiza o mesmo processo de resolver o sistema de equações para encontrar as novas coordenadas.

Questão 4 - Autovalores e Autovetores

Considere a matriz \(A = \begin{pmatrix} 2 & 1 & 0 \\ 0 & 2 & 1 \\ 0 & 0 & 3 \end{pmatrix}\).

a) Encontre o polinômio característico \(p(\lambda) = \det(A - \lambda I)\).
Ver Resposta
$$ A - \lambda I = \begin{pmatrix} 2-\lambda & 1 & 0 \\ 0 & 2-\lambda & 1 \\ 0 & 0 & 3-\lambda \end{pmatrix} $$

O determinante de uma matriz triangular é o produto de suas entradas na diagonal principal.

O polinômio característico é \(p(\lambda) = (2-\lambda)^2(3-\lambda)\).

b) Calcule os autovalores de A.
Ver Resposta

Os autovalores são as raízes do polinômio característico, ou seja, os valores de \(\lambda\) para os quais \(p(\lambda)=0\).

\((2-\lambda)^2(3-\lambda) = 0\)

Os autovalores são \(\lambda = 2\) e \(\lambda = 3\).

c) Para o **menor** autovalor encontrado, determine o autoespaço associado e apresente uma base para ele.
Ver Resposta

O menor autovalor é \(\lambda = 2\). Resolvemos \((A - 2I)\vec{x} = \vec{0}\):

$$ \begin{pmatrix} 0 & 1 & 0 \\ 0 & 0 & 1 \\ 0 & 0 & 1 \end{pmatrix} \begin{pmatrix} x \\ y \\ z \end{pmatrix} = \begin{pmatrix} 0 \\ 0 \\ 0 \end{pmatrix} $$

Isso nos dá as equações: \(y=0\) e \(z=0\). A variável \(x\) é livre. O autovetor é da forma \(\vec{x} = (t, 0, 0) = t(1, 0, 0)\).

O autoespaço \(E_2\) é o conjunto de todos os múltiplos de \((1,0,0)\), e uma base para ele é \(\{(1,0,0)\}\).

d) Para o **maior** autovalor encontrado, determine o autoespaço associado e apresente uma base para ele.
Ver Resposta

O maior autovalor é \(\lambda = 3\). Resolvemos \((A - 3I)\vec{x} = \vec{0}\):

$$ \begin{pmatrix} -1 & 1 & 0 \\ 0 & -1 & 1 \\ 0 & 0 & 0 \end{pmatrix} \begin{pmatrix} x \\ y \\ z \end{pmatrix} = \begin{pmatrix} 0 \\ 0 \\ 0 \end{pmatrix} $$

O sistema \(\begin{cases} -x+y=0 \\ -y+z=0 \end{cases}\) implica \(x=y=z\). O autovetor é da forma \(\vec{x} = (t, t, t) = t(1, 1, 1)\).

Uma base para o autoespaço \(E_3\) é \(\{(1,1,1)\}\).

e) Escolha um autovetor \(\vec{v}\) não nulo associado ao **maior** autovalor e verifique que a relação \(A\vec{v} = \lambda\vec{v}\) é satisfeita.
Ver Resposta

Usaremos o autovetor \(\vec{v}=(1,1,1)\) da base de \(E_3\) e seu autovalor correspondente \(\lambda=3\).

Lado esquerdo: \(A\vec{v}\)

$$ A\vec{v} = \begin{pmatrix} 2 & 1 & 0 \\ 0 & 2 & 1 \\ 0 & 0 & 3 \end{pmatrix} \begin{pmatrix} 1 \\ 1 \\ 1 \end{pmatrix} = \begin{pmatrix} 2+1+0 \\ 0+2+1 \\ 0+0+3 \end{pmatrix} = \begin{pmatrix} 3 \\ 3 \\ 3 \end{pmatrix} $$

Lado direito: \(\lambda\vec{v}\)

$$ \lambda\vec{v} = 3\begin{pmatrix} 1 \\ 1 \\ 1 \end{pmatrix} = \begin{pmatrix} 3 \\ 3 \\ 3 \end{pmatrix} $$

Como \(A\vec{v} = \lambda\vec{v}\), a verificação foi bem-sucedida.
